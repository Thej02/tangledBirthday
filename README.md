# 🌸 Tangled Birthday Invitation
A magical, lantern-themed invitation project built for college submission.

## 🏰 Project Overview
This project uses a **Maven** structure to manage a front-end invitation website. It features:
* Floating CSS lanterns
* A dreamy "Tangled" aesthetic
* Responsive layout for mobile and desktop

## 🛠️ Tech Stack
* **Build Tool:** Maven
* **Languages:** HTML5, CSS3, JavaScript
* **Theme:** Rapunzel / Tangled Lantern Festival

## 📜 How to Run
1. Navigate to `src/main/resources/static/`
2. Open `index.html` in any web browser.
